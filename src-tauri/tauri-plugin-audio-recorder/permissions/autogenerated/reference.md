## Default Permission

Default permissions for the Audio Recorder plugin - allows all audio recording operations

#### This default permission set includes the following:

- `allow-start-recording`
- `allow-stop-recording`
- `allow-pause-recording`
- `allow-resume-recording`
- `allow-get-status`
- `allow-get-devices`
- `allow-check-permission`
- `allow-request-permission`
